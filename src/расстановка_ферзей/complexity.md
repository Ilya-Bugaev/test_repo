Временная сложнотсть преборного решения:
itertools.product(range(N), repeat=N): генерирует N^N расстановок.
Внешний цикл: O(N) итераций для проверки каждого ферзя.
Внутренний цикл: O(N) итераций для проверки пар ферзей.
Общая сложность: O(N^N × N²)

Временная сложность улучшенного  переборного решения:
Генерация перестановок:  O(n!).
Проверка перестановки  O(n^2).
Общая сложность: O(n! * n^2).
